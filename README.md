# YouTube Trending Video Analytics

## Project Overview

This project analyzes YouTube trending videos across different countries to identify patterns in popularity, engagement, categories, and sentiment.

## Objective

To uncover insights from trending YouTube videos by analyzing:

* Trending videos by country
* Most popular channels
* Views vs Likes relationship
* Video categories
* Sentiment of video titles

## Tools Used

* Python
* Pandas
* Matplotlib
* TextBlob
* VS Code

## Dataset

The dataset contains 161,470 YouTube trending videos with information such as:

* Video Title
* Channel Name
* Category ID
* Views
* Likes
* Comments
* Country
* Trending Date

## Analysis Performed

### 1. Country Analysis

Analyzed trending videos across countries.

**Finding:** United States has the highest number of trending videos.

### 2. Top Channels Analysis

Identified channels with the most trending videos.

**Finding:** The Late Show with Stephen Colbert was the most frequently trending channel.

### 3. Views vs Likes Analysis

Studied the relationship between views and likes.

**Finding:** Videos with higher views generally receive more likes.

### 4. Category Analysis

Analyzed the most common video categories.

**Finding:** Category ID 24 had the highest number of trending videos.

### 5. Sentiment Analysis

Performed sentiment analysis on video titles using TextBlob.

**Finding:** Most video titles had neutral sentiment.

## Key Insights

* US dominates trending video counts.
* Entertainment channels frequently appear in trending lists.
* Engagement increases with video popularity.
* Neutral titles are more common than positive or negative titles.
